Configuration

Require Common Lisp library

Require builtin libraries here. 

(require 'cl-lib)

Environment

My variables

My user defined variables. I prefix all my variables with "my" to ensure there are no name collisions with other other modes variables. 

(defvar my-org-dir (concat user-emacs-directory "org"))

Load Paths

Add "~/.emacs.d" to load-path 

(let ((default-directory user-emacs-directory))
  (normal-top-level-add-subdirs-to-load-path))

User Info

Explicitly set these. 

(setq user-full-name "Jing Hao Sherman Pay"
      user-mail-address "shermanpay1991@gmail.com")

Startup

Right now I open a basic "todo.org" file at startup. Consider creating a personalized org file for startup. 

(add-hook 'emacs-startup-hook
          (lambda () (find-file (concat my-org-dir "/todo.org"))))
(setq inhibit-startup-message t)

Basic stuff

Yes/No prompts

Change yes or no prompts to y or n prompts 

(defalias 'yes-or-no-p 'y-or-n-p)

Uniquify

Uniquify buffer names with "post-forward" style: /foo/bar/name will become name|/foo/bar 

(require 'uniquify)
(setq uniquify-buffer-name-style 'post-forward)

Editing text

subword-mode: Allows navigation and editing of "subwords"

show-paren-mode: Show matching parentheses

next-line-add-newlines: If non-nil 'next-line' inserts a newline

save-place-mode: Save previous editing location and always go back to it 

(global-subword-mode t)

(show-paren-mode 1)
(setq show-paren-style 'expression)

(setq next-line-add-newlines nil)

(save-place-mode 1)

Ansi Color codes

Enable Ansi color codes for compilation buffers 

(require 'ansi-color)

Debugging Emacs

(setq debug-on-error t)

Backups

Setting Auto-Save directory and Backup Directory 

(let ((tmp (concat user-emacs-directory "tmp")))
 (setq backup-directory-alist
       `((".*" . ,tmp)))
 (setq auto-save-file-name-transforms
       `((".*" ,tmp t))))

User Interface

Line and Column numbers

I will like to see line and column numbers for all files. 

(global-linum-mode t)
(column-number-mode t)

Bars

Don't like all the fancy bars 

(menu-bar-mode -1)                      
(menu-bar-no-scroll-bar)
(tool-bar-mode -1)

Echo Area

Uses echo area to display tooltips and echo keystrokes 

(tooltip-mode -1)
(setq echo-keystrokes 0.01)

Emacs Server

Start emacs server on port 1337. 

(setq server-port 1337)
(setq server-use-tcp t)
(setq server-host (system-name))
(server-start) ; Start the emacs server

Package Management

Package management is important! Add melpa-stable repository, which has stable community packages. 

(require 'package)
(add-to-list 'package-archives
      '("melpa-stable" . "https://stable.melpa.org/packages/") t)
(package-initialize)

use-package

Bootstrap and customize it. always-ensure: Ensure the package exists by downloading it if it does not exist. 

(unless (package-installed-p 'use-package)
  (package-refresh-contents)
  (package-install 'use-package))
(eval-when-compile (require 'use-package))
(require 'use-package-ensure)
(setq use-package-always-ensure t)

evil-mode

I use evil-mode for Vim emulation

evil-move-cursor-back: don't want to move cursor back after exiting insert Add more distinguishable colors for evil states 

(use-package evil
  :config (evil-mode 1)
  (setq evil-move-cursor-back nil)
  (setq evil-normal-state-cursor '("dim gray" box)
        evil-insert-state-cursor '("dim gray" bar)
        evil-emacs-state-cursor '("blue" bar)))

Flyspell

Enable flyspell mode for strings and comments 

(add-hook 'prog-mode-hook (lambda () (flyspell-prog-mode)))

Use chrome as the default browser. This assumes chrome is installed. 

(setq browse-url-generic-program "/usr/bin/google-chrome"
      browse-url-browser-function 'browse-url-generic)

Other Packages/Plugins

The packages here are not required, but they each have their use case or provide a whole new experience.

Helm

Amazing package that provides a whole new UX 

(use-package helm
  :config (helm-mode 1)
  (setq helm-M-x-fuzzy-match t
        helm-split-window-inside-p t)
  :bind ("M-x" . helm-M-x))

Projectile

Package for working with "projects" 

(use-package projectile
  :config (projectile-global-mode 1))

Undo tree mode

This makes undo/redo in emacs behave like a tree! 

(use-package undo-tree
  :config (global-undo-tree-mode 1))

Font

I am experimenting with the Hack font at the moment. The following will try to set the font if it exists. Installation of the font is a manual step at the moment. 

(set-frame-font (find-font (font-spec :name "Hack" :weight 'normal :slant 'normal)))
